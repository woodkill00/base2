# Digital Ocean API Scripts

This folder contains scripts and modules for interacting with the Digital Ocean API to automate container deployment and management based on the current workspace build.

## Contents
- API client scripts for Digital Ocean
- Usage instructions
- Environment variable requirements

## Getting Started
1. Ensure you have a valid Digital Ocean API token.
2. Set required environment variables in `.env` and document them in `.env.example`.
3. Use provided scripts to launch, manage, and inspect containers via Digital Ocean API.
